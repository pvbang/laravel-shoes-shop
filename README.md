### Install: (Run in Terminal)
```bash
cp .env.example .env
composer install 	
# or: composer update 
# or: composer install --ignore-platform-reqs  
npm install
npm run dev
create database: pvbshoesshop   # utf8_unicode_ci
php artisan migrate:fresh --seed
php artisan key:generate
php artisan serve
```
